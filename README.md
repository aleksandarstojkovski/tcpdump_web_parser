# tcpdump_web_parser

Prerequisites 
```
- tcpdump
- httpry
```

Install
```
apt install tcpdump httpry dos2unix -y
git clone https://github.com/aleksandarstojkovski/tcpdump_web_parser
cd tcpdump_web_parser
chmod +x *.sh
dos2unix *.sh
```
